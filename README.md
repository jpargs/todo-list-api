
# Desctiption
Simple Todo list API built using rails 

# Startup Guide

 ### Prerequisites
 - Ruby
 - Rails
 - postgresql

###  Installation
- Clone Project
- Bundle Install
- run `rake db:create` to create database
- run `rake db:migrate` to add the columns
- run the server via `rails s`
